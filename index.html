How to Use

Stopwatch Instructions:

The stopwatch allows you to measure time with precision. Here's how to use it:

  1. Start: Press the Start button to begin the stopwatch.
  2. Stop: While the stopwatch is running, press Stop to pause the timer.
  3. Reset: Press the Reset button to set the timer back to 00:00:00.00.

Lap Timer Instructions:

The lap timer helps you record multiple time intervals or "laps" during a single session.

  1. Start: Press the Start button to begin tracking time.
  2. Lap: While the timer is running, press the Lap button to record a lap.
    • Each lap is displayed in the list below with the recorded time.
  3. Stop: Press the Stop button to pause the timer.
  4. Reset: Press the Reset button to clear the timer and all lap records.

Alarm Instructions:

The alarm feature allows you to set an alert for a specific time.

  1. Set Alarm:
    • Use the time input field to select the desired alarm time (in HH:MM format).
    • Press Set Alarm to activate the alarm.
    • The alarm time will be displayed along with a confirmation message.
  2. Cancel Alarm: If the alarm is active, press Cancel Alarm to deactivate it.
  3. Alarm Alert:
    • When the set time is reached, the alarm will display a message ("ALARM!") and play a sound.
    • To stop the alarm, press the Cancel Alarm button.
